## Description

Projeto feito em React Native Expo para funcionando tanto para web e mobile.

## Instalação

Para instalar as dependências do projeto, utilize o comando:

```bash
$ npm install
```

## Execução

Para rodar o aplicativo localmente, utilize o comando:

```bash
# development
$ npx expo start
```
